# PRD：商业化与增长指标

## 套餐阶梯

| 套餐 | 目标用户 | 计费方式 | 核心权益 |
| --- | --- | --- | --- |
| 免费试用 | 单个教师 | 免费额度 | 每月有限生成次数、基础导出；MVP-B 开放带品牌分享页 |
| 教师 Pro | 高频教师 | 月付 / 年付 | 更多生成次数、无水印、模板包、批量导出、更多局部重写；高质量 PPTX 属于标准版 |
| 机构版 | 学校 / 教培机构 | 按席位 + 用量 / 年度授权 | 统一模型池、额度、审计、校本模板、组织资源库、私有化部署 |

## 升级触发点

| 触发点 | 升级建议 |
| --- | --- |
| 免费生成次数用完 | 升级教师 Pro 获得更多生成次数 |
| 导出带水印 | 升级教师 Pro 去除水印 |
| 需要批量生成多个班级版本 | 升级教师 Pro 或机构版 |
| 需要校本模板 | 联系机构版 |
| 需要统一模型池、Key 和额度管理 | 机构版 |
| 需要私有化部署或本地模型 | 机构版 |

## MVP 增长漏斗

```text
访问首页
  → 查看示例
  → 上传课件
  → 生成成功
  → 首次预览
  → 局部重写
  → MVP-A 导出 / MVP-B 分享
  → 注册 / 付费
  → 第二次复用
```

## 核心指标

| 指标 | MVP 目标 |
| --- | --- |
| 基准输入首次有意义预览时间 | 热运行 P95 ≤ 3 分钟 |
| 首次成功生成用户 7 天内导出率 | ≥ 40% |
| 按质量标准判定无需大改比例 | ≥ 60% |
| 首次成功生成用户 7 天内局部重写比例 | ≥ 30% |
| 首次体验无需配置 API Key | 100% |
| 分享链接创建率 | 持续提升 |
| 学生自测完成率 | 持续提升 |
| 第二次复用率 | 持续提升 |

“持续提升”指标必须在首个真实用户月建立基线，此后按月报告绝对值和样本量；在没有基线前不作为发布门禁。指标事件、窗口和去重方式见 [埋点事件字典](../reference/event-tracking-dictionary.md)，质量与性能定义见 [MVP-A 测试与验收规范](../testing/mvp-a-acceptance.md)。
