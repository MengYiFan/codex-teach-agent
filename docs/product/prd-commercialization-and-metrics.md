# PRD：商业化与增长指标

## 套餐阶梯

| 套餐 | 目标用户 | 计费方式 | 核心权益 |
| --- | --- | --- | --- |
| 免费试用 | 单个教师 | 免费额度 | 每月有限生成次数、基础导出、带品牌分享页 |
| 教师 Pro | 高频教师 | 月付 / 年付 | 更多生成次数、无水印、模板包、批量导出、更多局部重写、高质量 PPTX |
| 机构版 | 学校 / 教培机构 | 按席位 + 用量 / 年度授权 | 统一模型池、额度、审计、校本模板、组织资源库、私有化部署 |

## 升级触发点

| 触发点 | 升级建议 |
| --- | --- |
| 免费生成次数用完 | 升级教师 Pro 获得更多生成次数 |
| 导出带水印 | 升级教师 Pro 去除水印 |
| 需要批量生成多个班级版本 | 升级教师 Pro 或机构版 |
| 需要校本模板 | 联系机构版 |
| 需要统一模型池、Key 和额度管理 | 机构版 |
| 需要私有化部署或本地模型 | 机构版 |

## MVP 增长漏斗

```text
访问首页
  → 查看示例
  → 上传课件
  → 生成成功
  → 首次预览
  → 局部重写
  → 导出 / 分享
  → 注册 / 付费
  → 第二次复用
```

## 核心指标

| 指标 | MVP 目标 |
| --- | --- |
| 上传到首版结果可预览时间 | ≤ 3 分钟 |
| 首次用户完成导出率 | ≥ 40% |
| 生成结果无需大改即可使用比例 | ≥ 60% |
| 用户完成一次局部重写比例 | ≥ 30% |
| 首次体验无需配置 API Key | 100% |
| 分享链接创建率 | 持续提升 |
| 学生自测完成率 | 持续提升 |
| 第二次复用率 | 持续提升 |
