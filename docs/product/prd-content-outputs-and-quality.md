# PRD：内容产物、审核与质量

## MVP-A 内容产物

| 产物 | 目标用户 | 价值 | 可编辑/重写 |
| --- | --- | --- | --- |
| 学生复习要点 | 教师、学生 | 快速发给学生复习 | 是 |
| 随堂小测与答案解析 | 教师、学生 | 课堂检测或课后自测 | 是 |
| 教师讲解提示 | 教师 | 帮助老师组织课堂语言 | 是 |
| 教案初稿 | 教师、教研员 | 形成可修改的备课文档 | 是 |

## 审核面板

结果页必须提供可扫描、可定位、可处理的审核面板，但只暴露高价值风险，避免增加老师负担。

### 顶部摘要

- 本次生成有多少处需要确认。
- 有多少处低置信内容。
- 有多少道题建议复核答案。

### 风险项内容

每个风险项展示：

1. AI 生成内容。
2. 来源页码或原文片段。
3. 风险原因。
4. MVP-A 操作：保留、修改、重新生成、删除、确认、保存为新版本；MVP-B 增加“确认并发布给学生”。

## 高风险类型

1. 题目答案可能不一致。
2. 来源页码缺失或识别不清。
3. AI 推断了年级、课时、学生水平等上下文。
4. MVP-B 发布给学生前仍处于未确认状态的内容。

## 审核能力分层

| 阶段 | 能力 |
| --- | --- |
| MVP-A | 来源缺失检查、低置信上下文提示、题型结构检查、单选答案唯一性、教师确认 |
| MVP-B | 发布前门禁、分享内容汇总检查、分享撤回与访问审计 |
| 标准版 | 跨内容答案一致性、学科事实审校、难度评估、批量质量报告 |
