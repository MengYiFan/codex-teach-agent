# 内容质量评分标准

## 评分方法

每个产物由至少一名具备对应学科教学经验的评审者按 1-5 分评分。发布基线评估时，20% 样本双人独立评分；加权 Kappa 低于 0.6 时先校准评分示例再继续。

| 分数 | 通用定义 |
| --- | --- |
| 1 | 严重错误或不可用，可能误导教学 |
| 2 | 多处关键问题，需要结构性重写 |
| 3 | 基本可用，但需要明显修改或补充 |
| 4 | 小改可用，不影响主要教学目标 |
| 5 | 可直接使用，准确、清晰且适配场景 |

## 教学质量

| 维度 | 权重 | 评分依据 | 自动检查 |
| --- | ---: | --- | --- |
| 事实与答案准确性 | 30% | 概念、公式、答案、解析和因果关系正确 | 单选唯一、答案格式、可计算题复算；人工判断事实 |
| 来源支持 | 20% | 关键结论能回到正确页码/片段，不伪造来源 | source ref 存在、页码有效、引用片段匹配 |
| 难度与年级匹配 | 15% | 术语、题目、例子符合已确认的年级和学生水平 | 规则检查 + 人工评分 |
| 教学结构与可讲授性 | 15% | 目标、重点、活动和时间组织连贯 | 结构完整性；人工评分 |
| 清晰性与可学习性 | 10% | 表达简洁，无歧义，学生能按步骤理解 | 可读性仅作提示；人工评分 |
| 覆盖与取舍 | 10% | 覆盖主要知识点，不过度扩展课件没有支持的内容 | CourseIR 覆盖率 + 人工评分 |

硬失败条件：任何关键答案错误、伪造来源、明显不适龄内容、泄露个人敏感数据或 blocking finding 未解决。存在硬失败时，无论平均分多少，该样本都不通过。

## 分产物检查

| 产物 | 附加要求 |
| --- | --- |
| 学习要点 | 重点明确、长度适中、术语首次出现有解释、易错点有依据 |
| 随堂小测 | 题干自洽、单选唯一、答案与解析一致、难度分布符合用途 |
| 教师讲解提示 | 不把不确定推断写成事实，包含可执行提问或过渡语 |
| 教案初稿 | 目标可观察、过程与课时匹配、活动和评价相互对应 |

## “无需大改”定义

同时满足以下条件才记为“无需大改即可使用”：

1. 教师在首次导出前选择“可直接使用”或“小改可用”。
2. 没有删除/新增一级结构，且人工编辑字符比例不超过 20%。
3. 没有硬失败，教学质量总分至少 4.0/5，准确性维度至少 4 分。

行为指标“是否导出、是否二次使用”只衡量产品价值，不能代替内容质量评分。

## MVP-A 发布阈值

| 指标 | 门槛 |
| --- | --- |
| 硬失败率 | ≤ 2%，且不得连续两个版本恶化 |
| 教学质量平均分 | ≥ 4.0/5 |
| 准确性平均分 | ≥ 4.2/5 |
| 有效来源覆盖率 | ≥ 90% 的关键内容块 |
| Schema 首次通过率 | ≥ 95%；自动修复后 ≥ 99% |
| 单选题答案/解析一致率 | ≥ 98% |

低于门槛时不得用单纯 Prompt 重试掩盖，应记录失败类型并修复解析、Schema、Prompt、模型路由或产品边界。
