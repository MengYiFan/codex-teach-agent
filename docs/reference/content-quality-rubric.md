# 内容质量评分标准

## 评分方法

每个请求产物（一个明确 OutputVersion）由至少一名具备对应学科教学经验的评审者按 1-5 分评分。发布基线评估时，20% 产物双人独立评分；加权 Kappa 低于 0.6 时先校准评分示例并重评受影响样本，再继续发布判定。

| 分数 | 通用定义 |
| --- | --- |
| 1 | 严重错误或不可用，可能误导教学 |
| 2 | 多处关键问题，需要结构性重写 |
| 3 | 基本可用，但需要明显修改或补充 |
| 4 | 小改可用，不影响主要教学目标 |
| 5 | 可直接使用，准确、清晰且适配场景 |

## 教学质量

| 维度 | 权重 | 评分依据 | 自动检查 |
| --- | ---: | --- | --- |
| 事实与答案准确性 | 30% | 概念、公式、答案、解析和因果关系正确 | 单选唯一、答案格式、可计算题复算；人工判断事实 |
| 来源支持 | 20% | 关键结论能回到正确页码/片段，不伪造来源 | source ref 存在、页码有效、引用片段匹配 |
| 难度与年级匹配 | 15% | 术语、题目、例子符合已确认的年级和学生水平 | 规则检查 + 人工评分 |
| 教学结构与可讲授性 | 15% | 目标、重点、活动和时间组织连贯 | 结构完整性；人工评分 |
| 清晰性与可学习性 | 10% | 表达简洁，无歧义，学生能按步骤理解 | 可读性仅作提示；人工评分 |
| 覆盖与取舍 | 10% | 覆盖主要知识点，不过度扩展课件没有支持的内容 | CourseIR 覆盖率 + 人工评分 |

硬失败条件：任何关键答案错误、伪造来源、明显不适龄内容、泄露个人敏感数据或 blocking finding 未解决。存在硬失败时，无论平均分多少，该样本都不通过。

## 聚合公式

1. 单产物总分为六个维度分数乘对应权重之和，保留两位小数；硬失败产物仍记录维度分，但通过判定为失败。
2. 一个 generation run 请求多个产物时，各请求产物等权计算 run 平均分；请求但未生成的产物不虚构质量分，计入可靠性失败并使该 run 的整体质量判定失败。
3. 发布候选的教学质量和准确性平均分，以成功生成且完成评审的 OutputVersion 为分母，同时按产物类型分别报告，禁止用高分产物掩盖某一类型不达标。
4. 硬失败率 = 硬失败 OutputVersion 数 / 完成评审的 OutputVersion 数；任一请求产物硬失败时，该 generation run 的整体质量判定失败。报告同时给出产物级和 run 级比例。
5. 来源覆盖率 = 含至少一个有效且可回查 source ref 的关键内容块数 / 应有来源的关键内容块数；纯文本需求中明确属于教师原创约束的块可标记 `source_not_applicable`，但必须人工确认且单独报告。

## 分产物检查

| 产物 | 附加要求 |
| --- | --- |
| 学习要点 | 重点明确、长度适中、术语首次出现有解释、易错点有依据 |
| 随堂小测 | 题干自洽、单选唯一、答案与解析一致、难度分布符合用途 |
| 教师讲解提示 | 不把不确定推断写成事实，包含可执行提问或过渡语 |
| 教案初稿 | 目标可观察、过程与课时匹配、活动和评价相互对应 |

## “无需大改”定义

同时满足以下条件才记为“无需大改即可使用”：

1. 教师或受控评审者在首次导出前，对明确版本清单选择“可直接使用”或“小改可用”。
2. 没有删除/新增一级结构，且人工编辑字符比例不超过 20%。字符比例在 Unicode NFC、统一换行并折叠连续空白后计算：`min(1, Levenshtein(首个模型可用版本纯文本, 首次导出版本纯文本) / max(1, 首个模型可用版本字符数))`；标题和正文计入，纯样式变化不计入。
3. 没有硬失败，教学质量总分至少 4.0/5，准确性维度至少 4 分。

行为指标“是否导出、是否二次使用”只衡量产品价值，不能代替内容质量评分。

## MVP-A 发布阈值

| 指标 | 门槛 |
| --- | --- |
| 硬失败率 | ≤ 2%，且不得连续两个版本恶化 |
| 教学质量平均分 | ≥ 4.0/5 |
| 准确性平均分 | ≥ 4.2/5 |
| 有效来源覆盖率 | ≥ 90% 的关键内容块 |
| Schema 首次通过率 | ≥ 95%；自动修复后 ≥ 99% |
| 单选题答案/解析一致率 | ≥ 98% |

低于门槛时不得用单纯 Prompt 重试掩盖，应记录失败类型并修复解析、Schema、Prompt、模型路由或产品边界。

“连续两个版本恶化”只比较相同固定核心集和评分流程。若教学质量平均分或准确性平均分下降至少 0.10，或硬失败率上升至少 1 个百分点，记为一次恶化；样本变化或评分者校准变化必须单独重建基线，不能直接串联比较。任一指标低于绝对门槛仍直接失败，不等待连续两个版本。
