# ADR 0002：CourseIR 作为内部中心模型

## 状态

Accepted

## 背景

系统需要从 PPTX/PDF 等输入生成学习要点、随堂小测、教师提示、教案、分享页和导出文件。如果每个产物直接从原始文件生成，会导致来源追溯、版本管理和局部重写困难。

## 决策

使用 CourseIR 作为内部统一课程中间表示。教师端不显示 CourseIR，统一展示为“课程结构 / 知识点大纲”。

## 影响

- 所有产物可以共享知识点、章节和来源页码。
- 局部重写和内容块版本可以追溯到 CourseIR。
- 需要维护 CourseIR Schema、fixtures 和迁移策略。
